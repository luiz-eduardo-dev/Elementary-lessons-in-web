Tags Descontinuadas no HTML 5

Inutilizadas:

As tags abaixo foram inutilizadas devido ao fato de serem melhor configuradas por CSS:

  • <basefont> (Define as propriedads da font padrão para todo o texto do documento).
  • <big> (Usado para tornar o texto maior).
  • <center> (Usado para alinhar texto e conteúdo no centro).
  • <font> (Especifica o tipo de fonte, tamanho, e cor do texto).
  • <strike> (Exibe texto rasurado).
  • <tt> (Define teletipo de texto).
  • <u> (Define o texto sublinado).

Removidas:

Os elementos abaixo foram removidos por prejudicar a navegação e usabilidade dos usuários:

  • <frame> (Define uma janela particular dentro de um conjunto de "frames").
  • <frameset> (Define um conjunto de frames organizado por múltiplas janelas).
  • <noframes> (Texto exibido para navegadores que não lidam com "frames").

Substituídas:

Os elementos abaixo foram substituidos por outros elementos por serem pouco utilizados.

  • <acronym> (Define siglas em HTML 4.01), foi substituido por <abbr>.
  • <applet> (Define um miniaplicativo incorporado), foi substituido por <object>.
  • <dir> (Define a lista do diretório), foi substituido por <ul>.

Apesar dessas tags serem obsoletas, o HTML5 ainda reconhece o funcionamento dessas tags. Como podemos observar, todas elas foram substituidas por outras, então você não tera problemas de compatibilidade ao utilizar tags das outras versões de HTML.
